ArrayAccess::offsetUnset

Unset an offset

Description

abstract public void ArrayAccess::offsetUnset ( mixed $offset )

Unsets an offset.

Note:

This method will not be called when type-casting to (unset)

Parameters

offset

The offset to unset.

Return Values

No value is returned.
