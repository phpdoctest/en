ArrayIterator::ksort

Sort array by keys

Description

public void ArrayIterator::ksort ( void )

Sorts an array by the keys.

Warning

This function is currently not documented; only its argument list is available.

Parameters

This function has no parameters.

Return Values

No value is returned.
