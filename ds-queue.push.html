Ds\Queue::push

Pushes values into the queue.

Description

public void Ds\Queue::push ([ mixed $...values ] )

Pushes values into the queue.

Parameters

values

The values to push into the queue.

Return Values

No value is returned.

Examples

Example #1 Ds\Queue::push example

<?php
$queue = new \Ds\Queue();

$queue->push("a");
$queue->push("b");
$queue->push("c", "d");
$queue->push(...["e", "f"]);

print_r($queue);
?>

The above example will output something similar to:

Ds\Queue Object
(
    [0] => a
    [1] => b
    [2] => c
    [3] => d
    [4] => e
    [5] => f
)
