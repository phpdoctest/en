Ds\Set::diff

Creates a new set using values that aren't in another set.

Description

public Ds\Set Ds\Set::diff ( Ds\Set $set )

Creates a new set using values that aren't in another set.

A \ B = {x ∈ A | x ∉ B}

Parameters

set

Set containing the values to exclude.

Return Values

A new set containing all values that were not in the other set.

See Also

  • » Compliment on Wikipedia

Examples

Example #1 Ds\Set::diff example

<?php
$a = new \Ds\Set([1, 2, 3]);
$b = new \Ds\Set([3, 4, 5]);

var_dump($a->diff($b));
?>

The above example will output something similar to:

object(Ds\Set)#3 (2) {
  [0]=>
  int(1)
  [1]=>
  int(2)
}
