Event::addTimer

Makes timer event pending

Description

public bool Event::addTimer ([ double $timeout ] )

Event::addTimer is an alias of Event::add

Return Values

Examples

Example #1 Event::addTimer example

<?php
$base = new EventBase();
$n = 2;
$e = Event::timer($base, function($n) use (&$e) {
    echo "$n seconds elapsed\n";
    $e->delTimer();
}, $n);
$e->addTimer($n);
$base->loop();
?>

The above example will output something similar to:

2 seconds elapsed
