KTaglib_Tag::getYear

Returns the year from a ID3 tag

Description

public int KTaglib_Tag::getYear ( void )

Returns the year of an ID3 tag. This method is implemented in ID3v1 and ID3v2 tags.

Return Values

Returns the year as an integer
