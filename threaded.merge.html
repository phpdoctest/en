Threaded::merge

Manipulation

Description

public boolean Threaded::merge ( mixed $from [, bool $overwrite ] )

Merges data into the current object

Parameters

from

The data to merge

overwrite

Overwrite existing keys, by default true

Return Values

A boolean indication of success

Examples

Example #1 Merging into the property table of a threaded object

<?php
$array = [];

while (count($array) < 10)
    $array[] = count($array);

$stdClass = new stdClass();
$stdClass->foo = "foo";
$stdClass->bar = "bar";
$stdClass->baz = "baz";

$safe = new Threaded();
$safe->merge($array);

$safe->foo = "bar";
$safe->merge($stdClass, false);

var_dump($safe);
?>
