tidy::root

tidy_get_root

Returns a tidyNode object representing the root of the tidy parse tree

Description

Object oriented style

tidyNode tidy::root ( void )

Procedural style

tidyNode tidy_get_root ( tidy $object )

Returns a tidyNode object representing the root of the tidy parse tree.

Parameters

object

The Tidy object.

Return Values

Returns the tidyNode object.

Examples

Example #1 tidy::root example

<?php

$html = <<< HTML
<html><body>

<p>paragraph</p>
<br/>

</body></html>
HTML;

$tidy = tidy_parse_string($html);
dump_nodes($tidy->root(), 1);


function dump_nodes($node, $indent) {

    if($node->hasChildren()) {
        foreach($node->child as $child) {
            echo str_repeat('.', $indent*2) . ($child->name ? $child->name : '"'.$child->value.'"'). "\n";

            dump_nodes($child, $indent+1);
        }
    }
}
?>

The above example will output:

..html
....head
......title
....body
......p
........"paragraph"
......br

Notes

Note: This function is only available with Zend Engine 2 (PHP >= 5.0.0).
