Yaf_Route_Rewrite::assemble

Assemble a url

Description

public string Yaf_Route_Rewrite::assemble ( array $info [, array $query ] )

Assemble a url.

Parameters

info

query

Examples

Example #1 Yaf_Route_Rewrite::assembleexample

router = new Yaf_Router();

$route  = new Yaf_Route_Rewrite(
                "/product/:name/:id/*",
                array(
                        'controller' => "product",
                ),
                array()
);

$router->addRoute("rewrite", $route);

var_dump($router->getRoute('rewrite')->assemble(
                        array(
                                ':name' => 'foo',
                                ':id' => 'bar',
                                ':tmpkey1' => 'tmpval1'
                        ),
                        array(
                                'tkey1' => 'tval1',
                                'tkey2' => 'tval2'
                             )
                        )
);

The above example will output something similar to:

string(57) "/product/foo/bar/tmpkey1/tmpval1/?tkey1=tval1&tkey2=tval2"
